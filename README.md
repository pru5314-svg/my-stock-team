# my-stock-team 📈

종목명 하나를 입력하면 애널리스트(서브에이전트)들이 협업해 리서치를 만들고,
최종 결과를 **PPTX 리포트**로 내보내는 자동 주식 리서치 시스템입니다.

> ⚠️ 실습용 더미 프로젝트입니다. 본 리포트는 투자 권유가 아니며, 투자 판단의 책임은 본인에게 있습니다.

## 워크플로우

```
종목명 입력
  → 데이터 수집 (시세·재무·뉴스)
  → 애널리스트 협업 분석 (기술 / 펀더멘털 / 뉴스·심리)   ← 병렬 가능
  → 리스크 종합 (risk-manager)                          ← 위 3개 결과 필요 (순차)
  → 리서치 취합·검수 (research/)
  → PPTX 리포트 생성 (reports/)
```

## 애널리스트 (서브에이전트)

`.claude/agents/`에 정의된 4종. 각자 독립 데이터 소스를 가집니다.

| 에이전트 | 역할 | 데이터 소스 | 의존성 |
|---------|------|------------|--------|
| `market-technical-analyst` | 주가·추세·거래 동향 | FinanceDataReader (키 불필요) | 없음 (독립) |
| `fundamental-analyst` | 재무·실적·공시 | DART (`.env`의 `DART_KEY`) | 없음 (독립) |
| `news-sentiment-analyst` | 뉴스·이슈·시장 심리 | Claude Code 웹서치 (키 불필요) | 없음 (독립) |
| `risk-manager` | 결과 종합·리스크 점검 | pykrx (키 불필요) + 위 3개 결과 | **위 3개 완료 후** |

공통 규칙: 매수/매도·목표가 단정 금지, 모든 수치에 출처·기준일 표기, 한국어 출력, 면책 문구 포함.

## 폴더 구조

| 폴더 | 역할 |
|------|------|
| `.claude/agents/` | 애널리스트 서브에이전트 |
| `.claude/skills/` | 재사용 스킬 (PPTX 생성 등) |
| `.claude/commands/` | 슬래시 커맨드 (예: `/research <종목>`) |
| `src/` | 데이터 수집·분석·리포트 생성 코드 |
| `data/` | 원본 / 가공 / 캐시 데이터 |
| `research/` | 애널리스트 중간 산출물 (종목별) |
| `reports/` | 최종 PPTX 리포트 출력물 |
| `templates/` | PPTX 템플릿·슬라이드 레이아웃 |
| `config/` | 설정 (종목 목록 등) |
| `tests/` | 테스트 |

## 설치

```bash
pip install -r requirements.txt
```

## 환경 변수

`.env.example`을 복사해 `.env`로 만들고 값을 채우세요.

```bash
cp .env.example .env
```

| 키 | 설명 | 발급처 |
|----|------|--------|
| `DART_KEY` | 금융감독원 OPEN DART API 키 | https://opendart.fss.or.kr (무료) |

> `.env`는 `.gitignore`로 제외되어 커밋되지 않습니다.

## 작업 규칙

- **데이터로 말한다** — 근거 없는 추측 금지. 수치·출처를 명시한다.
- **편향 없이** — 매수/매도 한쪽으로 치우치지 않고 리스크를 함께 제시한다.
- **투자 권유 아님** — 모든 리포트에 면책 문구 필수.
- **출력은 한국어** — 리서치·리포트·커뮤니케이션 모두 한국어.
