- DART 고유번호 **00126380**. 본 리포트의 재무·공시 수치는 모두 금융감독원 전자공시(DART) Open API **연결재무제표(CFS)** 기준이다.
- 실적 사이클의 핵심 동인은 **메모리 반도체 업황(DRAM·NAND 가격)**으로, 2023년 메모리 다운사이클 → 2024~2025년 회복 구간을 거쳤다.

## 리스크
1. **메모리 업황 의존** — 2023년 영업이익률 2.5%로 급락한 사례처럼 DRAM·NAND 가격 사이클에 이익이 크게 좌우된다. / 모니터링: 분기 메모리 가격·출하 추이 (근거: DART 2023 연결, 영업이익 6.57조)
2. **부채 증가 추세** — 부채총계가 92조(2023)→131조(2025)로 증가, 부채비율 25%→30%. 절대 수준은 안정적이나 추세는 점검 필요. / 모니터링: 분기 부채비율·이자비용 (근거: DART 연결 재무상태표)
3. **데이터 신뢰성** — 2026년 1분기 손익 수치에 이상치가 확인됨. 후속 분기 보고서로 재검증 전까지 분기 실적 해석에 유의. / 모니터링: DART 정정공시·차기 분기보고서 (근거: DART 11013, 1Q OP 57.23조 이상치)
