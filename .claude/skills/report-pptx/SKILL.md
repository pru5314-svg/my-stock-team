# report-pptx — 리서치 PPTX 생성 스킬

종목 리서치 마크다운을 투자증권 리서치센터 양식의 PPTX로 변환한다.

## 사용법
```bash
# 프로젝트 루트(Desktop/stock)에서 실행
python .claude/skills/report-pptx/scripts/build_pptx.py "삼성전자"
```
- 입력: `reports/{종목명}.md`
- 출력: `reports/pptx/{종목명}.pptx` (`reports/pptx/` 폴더가 없으면 생성)
- 옵션: `--input <경로>` `--output <경로>` `--date YYYY-MM-DD` 로 개별 지정 가능.

### 의존성
- `python-pptx` 필요. 미설치 시: `pip install python-pptx`
- (이미지 차트는 마크다운이 참조하는 PNG 파일을 그대로 임베드한다. 별도 차트 라이브러리 불필요.)

## 입력 마크다운 규약 (이 형식으로 작성)
슬라이드는 **항상 아래 순서로 고정**된다. 스크립트는 `##` 섹션 제목의 키워드로 내용을 각 슬라이드에 매핑한다(누락 섹션은 "내용 없음" 플레이스홀더로 표시).

```markdown
# 삼성전자                      ← 표지 종목명 (첫 H1)
작성일: 2026-06-16             ← 표지 작성일 (없으면 오늘 날짜)

## 종목 개요
- 한 줄 소개, 사업 구조 등

## 재무 요약
| 항목 | 2023 | 2024 | 2025 |
|------|------|------|------|
| 매출액 | ... | ... | ... |   ← 표가 있으면 표로 렌더 (출처/기준일 열·문구 포함)

## 가격/추세
![차트](../assets/charts/삼성전자.png)   ← 이미지 참조 시 임베드
| 항목 | 값 |  ← 또는 요약표

## 뉴스·심리
1. (이슈 한 줄) — 출처: <링크> (YYYY-MM-DD)
- 시장 심리: 중립

## 리스크
1. (리스크) — 근거: ... / 모니터링: ...

## 한 줄 종합
- (사실 기반 종합 한 줄)
```

섹션 제목 매핑 키워드: 개요/기업 · 재무/실적 · 가격/추세/주가/기술 · 뉴스/심리/센티 · 리스크/위험 · 종합/결론.

## 디자인 (투자증권 리서치 톤 — 스크립트에 고정)
- 배경: 화이트
- 제목: 딥네이비 `#1F2A44`
- 본문: 그레이
- 한화 오렌지 `#F37321`: **표지 상단 바 · 제목 아래 구분선 · 키 수치 강조**에만 절제해서 사용
- 표: 헤더행만 딥네이비 배경 + 흰 글자, 본문행은 얇은 그레이 괘선(`#D9D9D9`)
- 한글 폰트: **맑은 고딕** 하나로 고정(latin·ea·cs 모두 지정 → 글자 깨짐 방지)

### 키 수치 강조
본문/셀에서 `**값**` (마크다운 볼드)로 감싼 부분은 **오렌지 볼드**로 강조된다. 핵심 수치에만 절제해서 사용한다.

## 콘텐츠 규칙 (엄수 — 스크립트가 린트로 경고)
- **모든 수치 옆에 (출처/기준일) 표기.** 출처 없는 숫자는 싣지 않는다.
- **매수/매도·목표가 단정 표현 금지.** 판단 근거까지만 제시한다.
  - 스크립트는 `매수/매도/목표가/비중확대/BUY/SELL` 등 금지어를 발견하면 경고를 출력한다. 경고가 나오면 마크다운을 수정한 뒤 다시 생성한다.
- 마지막 슬라이드/표지에 면책 문구(투자 판단 책임은 본인) 포함.

## 작업 절차
1. `reports/{종목명}.md`가 위 규약대로 작성됐는지 확인(없으면 애널리스트 산출물을 취합해 먼저 작성).
2. 스크립트 실행 → `reports/pptx/{종목명}.pptx` 생성.
3. 린트 경고(금지어·출처 누락)가 있으면 마크다운을 고치고 재실행.
